RECENT PROJECTS

Below is the list of selected projects completed in the recent months

MAUSER PACKAGING, Chicago, IL, USA

Upgrade of multiple gravimetric blenders by installing WXLB control system on each individual blender. The WXLB system is designed to work with a single blender and handles up to 8 components.
The project resulted in significant increase in throughput and accuracy. Our proprietary "HIGH ACCURACY" blending mode allowed to dose expensive resins with superior accuracy. The recorded process statistics helped establish a modern product quality control process.

POLYTARP PRODUCTS, Toronto, ON, CANADA

INTELECS™ BATCH installation integrating 4 large Mould-Tek 8-component gravimetric blenders. All basic features such as alarm handling, local recipe storage, user-based security, material consumption reports and material database are included.
The control system also handles 30 loaders serviced by 4 pumps and manages resin distribution from central material storage silos to each blender.
The system is connected to the INTELECS™ HISTORIAN server that stores results for each batch in a relational database and provides software tools for data visualization and analysis. A serial printer is attached directly to the controller for printing the reports. A second INTELECS™ BATCH controller is supplied to provide full system redundancy.

ALTE-REGO CORPORATION, Toronto, ON, CANADA

Upgrade of multiple Mould-Tek gravimetric blenders by installing WXLB control system on each individual blender. The WXLB system is designed to work with a single blender and handles up to 8 components.
